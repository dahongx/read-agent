# 文献阅读

## 需求功能

1. **自动读论文**
2. **生成 PPT**
3. **生成口头讲稿**
4. **语音播报**
5. **基于文献数据库进行语音互动**

## 详细功能

1. **自动读论文：上传几篇文献自动总结文献要点 （v2 也能够根据关键词检索下载文献，能够根据 url 自动爬取文件）**
2. **生成 PPT：根据文献要点，生成 PPT（doc to ppt）**
3. **生成口头讲稿：根据 PPT 生成口头讲稿（TTS）**
4. ** 语音播报：一个网页，左边显示 ppt，右边显示问答界面（这个界面可以参考 柴东东做的 AI 课堂）**
5. **基于文献数据库进行语音互动：用户举手提问，识别语音，检索知识库（文献），结合知识库进行 LLM 回答，TTS 合成回答内容播放。要能定位到文献原文，最好是答案上有个链接 可点击跳转到文献原文位置。**
